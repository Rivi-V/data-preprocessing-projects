# Data Analysis and Preprocessing

В этом репозитории собраны учебные mini-projects по анализу данных, визуализации и базовым ML-подходам. Основной акцент сделан на работе с `pandas`, `matplotlib`, `seaborn`, исследовании данных и построении простых baseline-моделей.

## Проекты

### `eda_students_performance.ipynb`
Разведочный анализ датасета **StudentsPerformance**.  
В ноутбуке рассматриваются структура данных, основные статистики, группировки по категориям и простые аналитические выводы по результатам учащихся.

### `eda_visualization_students.ipynb`
Ноутбук с акцентом на визуализацию данных из датасета **StudentsPerformance**.  
Содержит гистограммы, диаграммы рассеяния, boxplot и heatmap для анализа распределений и зависимостей между признаками.

### `user_behavior_analysis.ipynb`
Анализ пользовательского поведения на платформе **Stepik**.  
В проекте используются события пользователей и данные об отправках решений, создаются признаки для ML и обучается baseline-модель для задачи классификации.

## Что используется

- scikit-learn

## Цель репозитория

Показать базовые навыки:
- работы с табличными данными;
- исследовательского анализа данных (EDA);
- визуализации;
- подготовки признаков;
